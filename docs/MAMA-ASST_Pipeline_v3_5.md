## 기술 스택

| 계층 | 기술 | 용도 |
|------|------|------|
| Frontend | Bubble.io | 노코드 웹 인터페이스 |
| Backend | Bubble.io Workflows | API 및 비즈니스 로직 |
| Database | Bubble DB | 데이터 저장 |
| AI | Gemini Flash / Pro | 콘텐츠 생성, 분석 |
| SMS/알림 | NCP SENS | SMS, 카카오 알림톡 |
| 결제 | 결제선생 (Paymint) | 청구서 발행, 결제 |
| STT | Web Speech API | 7초 발화 음성 인식 |
